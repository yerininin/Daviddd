  • Release/Reject 데이터 분석

Topic 04

Future Strategy

4.1 이커머스 확장 (Figma MCP 연동)

Claude Code에 Figma MCP를 연동하여, 브랜드별 상세페이지 규칙과 채널별 배너 규칙을 스킬(Skill)로 문서화하고 AI 이미지 생성부터 채널 배너 일괄 생성까지 E2E 자동화 파이프라인을 구축합니다.

Step 1

Figma MCP 연동

Claude Code에 Figma MCP를 연결하여 디자인 자동화 기반 구축

Step 2

커머스 규칙 스킬 문서화

브랜드별 상세페이지 규칙과 채널별 배너 사이즈/레이아웃 규칙을 스킬로 문서화

Step 3

E2E 자동화

AI 이미지 생성 → 상세페이지 세팅 → 채널 배너 일괄 생성 파이프라인

Figma MCP 연동 - 상세페이지 + 채널 배너 자동 생성 데모

4.2 지식그래프 연계

내부·외부 데이터를 지식그래프로 연계하여 최적 프롬프트를 자동 선택하고, 이커머스까지 자동 세팅합니다.

  • 데이터 기반 자동 선택

  • 이미지 생성 + 검수
  • 상세페이지 자동 세팅
  • 채널 배너 일괄 생성
